()-Event binding Sysntax is used to be pass the data to html to ts.
[]-Property Binding syntax is used to passed th data from ts to html.
[()]- This is called as banana in box syntax which is used for two way data binding.

But in order to use to way data binding we have to use a built in module known as FormsModule.
Two way data binding is very important function of angular and very popular.
All form submission is done using two way binding syntax.

[(ngModel)] = "username"
where ngModel is a built-in attribute given by Angular.
username is a field defined in the ts file.
